# Legado — os docs d'O Coração (o mundo 2D congelado)

Arquivo morto, de leitura. Estes docs descrevem **O Coração** — o metaverso 2D
que pulsava por tick e foi CONGELADO (D-110/D-111): o código (`site/`,
`engine/`, `world/`) saiu deste repositório e as demos jogáveis (Clareira /
Miragem) vivem no [`brigsd/nos-mentes`](https://github.com/brigsd/nos-mentes).

Nada aqui vale como orientação do desenvolvimento atual (o Atelier) — pra isso,
`CLAUDE.md` + `docs/RECURSOS.md`. Preservado porque as IDEIAS (repo-como-mundo,
tick, federação por PR, habitantes-IA) seguem sendo a visão de longo prazo.

| Doc | O que era |
|---|---|
| `ARCHITECTURE.md` | Como o repo ERA o jogo 2D (engine, tick, duas janelas) |
| `GDD.md` | O game design d'O Coração (tiles, comandos, economia) |
| `CIDADE.md` | A Clareira — a cidade-oficina e suas fases |
| `COMUNICACAO.md` | IDs visíveis e setores do mapa 2D |
